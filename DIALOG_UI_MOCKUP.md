## Visual Hierarchy

1. **Title & Description**: Top section, primary input
2. **Category & Priority**: Middle section, context settings
3. **Date & Time**: Horizontal layout, temporal settings
4. **Reminder**: Bottom section, notification settings
5. **Actions**: Dialog buttons at the bottom

## Responsive Design

- Spinners expand to full width on small screens
- Date/Time fields use horizontal layout with equal weight
- Icons scale proportionally with text size
- Touch targets maintain 48dp minimum height

## Accessibility Features

- Clear text labels for all fields
- Icons supplement but don't replace text
- Error messages appear inline
- Required fields marked with asterisk (*)
- Tab order follows logical flow
- Color is not the only indicator (icons + text)

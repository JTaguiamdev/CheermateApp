## Color Scheme & Styling

All elements use the app's existing design system:

- **Background:** Glass card effect with blur
- **Text Color:** White (#FFFFFF)
- **Toolbar Background:** Glass card background
- **Bottom Sheet Background:** `@drawable/bg_card_glass`
- **Fonts:** San Francisco Pro Rounded
- **Icons:** White with 24dp size
- **Emojis:** For visual clarity (âœ… â° âŒ)
- **Ripple Effect:** Material Design ripple on tappable items

## Animation Flow

1. **Bottom Sheet Entrance:**
   - Slides up from bottom edge
   - Duration: ~300ms
   - Easing: Material motion curve
   - Background dims slightly

2. **Bottom Sheet Exit:**
   - Slides down to bottom edge
   - Duration: ~200ms
   - Background returns to normal

3. **Toast Notification:**
   - Fades in from bottom
   - Displays for 2 seconds
   - Fades out

## Interaction States

### 3-Dot Icon
- **Normal:** White, 24dp
- **Pressed:** Ripple effect
- **Hover:** Slightly brighter (on tablets/chromebooks)

### Bottom Sheet Options
- **Normal:** White text on glass background
- **Pressed:** Ripple effect spreads from touch point
- **Selected:** Bottom sheet dismisses

### Confirmation Buttons
- **Cancel:** Neutral, dismisses dialog
- **Yes/OK:** Positive action, executes operation
